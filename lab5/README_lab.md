# Laboratório de Automação - Coleta de Inventário

Este projeto coleta automaticamente informações de dispositivos de rede (versão do IOS, interfaces, hardware) usando Ansible.

## Como usar

```bash
ansible-playbook -i inventario.yml playbook_inventario.yml
```
